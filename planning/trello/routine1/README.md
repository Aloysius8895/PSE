# Routine 1 (02/03/2026 ~ 04/03/2026) - Initiation and Planning： 
In the process, the team first discussed and made a decision regarding the topic for the project. After the topic was confirmed, the tasks were allocated to each member of the team based on their role. The team also created the GitHub repository and Trello board for the organization of the project information. The team used GitHub to store the project files and track changes to the versions of the files. The team also planned the timeline for the assigned tasks to ensure that each member of the team was able to fulfill their role within the allocated time frame. The above process helped to lay the foundation for the development process.
