# Tutor User Stories:
## Tutor Profile Management:

* User Story 1:  
As a tutor, I want to create a tutor profile so that students can find me on the platform.

* User Story 2:  
As a tutor, I want to upload a profile photo so that students can identify me.

* User Story 3:  
As a tutor, I want to edit my biography so that I can introduce my teaching experience.

* User Story 4:  
As a tutor, I want to set my hourly rate so that students know my tutoring price.

* User Story 5:  
As a tutor, I want to edit subjects so that I can specify the subjects I teach.

## Availability Management:

* User Story 6:  
As a tutor, I want to manage my availability so that students can book sessions at suitable times.

## Booking Management:

* User Story 7:  
As a tutor, I want to view booking requests so that I can manage tutoring sessions.

* User Story 8:  
As a tutor, I want to accept booking requests so that I can confirm tutoring sessions.

* User Story 9:  
As a tutor, I want to reject booking requests so that I can decline sessions when I am unavailable.
