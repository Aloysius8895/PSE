# Student User Stories

## Account Management：

* User Story 1:  
As a student, I want to create an account so that I can access the UniTutor platform.

* User Story 2:  
As a student, I want to log in to my account so that I can use the tutoring services.

## Tutor Discovery：

* User Story 3:  
As a student, I want to search for tutors so that I can find academic help for my subject.

* User Story 4:  
As a student, I want to filter tutors so that I can find tutors based on specific criteria.

* User Story 5:  
As a student, I want to view tutor profiles so that I can learn about their expertise and experience.

* User Story 6:  
As a student, I want to view tutor ratings so that I can choose high-quality tutors.

## Booking Sessions：

* User Story 7:  
As a student, I want to select a tutoring date so that I can schedule a tutoring session.

* User Story 8:  
As a student, I want to select a time slot so that I can book a convenient session.

* User Story 9:  
As a student, I want to check tutor availability so that I can ensure the tutor is free.

* User Story 10:  
As a student, I want to confirm a tutoring booking so that the tutoring session is scheduled.

* User Story 11:  
As a student, I want to view my booking status so that I know whether my booking is accepted or rejected.

## Communication：

* User Story 12:  
As a student, I want to send messages to tutors so that I can communicate with them before the session.

## Rating and Feedback:

* User Story 13:  
As a student, I want to rate tutors so that I can provide feedback about the tutoring session.

* User Story 14:  
As a student, I want to submit comments so that I can share detailed feedback with other students.
