# Testing Evidence

This folder contains the testing documentation for the Peer Tutoring Platform prototype.

The testing phase evaluates whether the main system functions work correctly and whether the user interface is easy to use.

Test cases are divided into two groups:
- Student test cases
- Tutor test cases

The tests include registration, login, tutor search, session booking, tutor profile management, and rating functionality.

All 7 functional tests were successfully passed during the evaluation of the prototype.
