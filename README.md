## Project Description:
* This project aims to develop a prototype of a Tutor Matching Platform that connects students with tutors based on subject, availability, and preferences. The platform is designed for students to find suitable tutors and schedule tutoring sessions more efficiently and easily
  
* The system allows tutors to create profiles and list their expertise for student as a reference while students search for tutors. Students can view tutor information, search for tutor that they needed and suitable for them, and also book tutoring session. By providing an well-organized platform for tutor discovery and session booking, the system can helps to improve access to educational support
